Installation
Copy the batch file(Choose any batch file you prefer) to custom folder and double-click the batch file. For Left 4 Dead 2, put batch file on addon folder. Press any key and you're done!

WARNING:
Cleaning cache files outside of custom/addons folder may cause instability.
